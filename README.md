# AEDN-Endocrine-Classification

Attention-Enhanced Endocrine Deep Network (AEDN) for Thyroid Disease and PCOS Classification.

## Datasets
- UCI Thyroid Disease Dataset
- Kaggle PCOS Clinical Dataset

## Models
- Logistic Regression
- SVM
- Random Forest
- CNN
- BiLSTM
- AEDN

## Features
- Missing Value Imputation
- Standardization
- Correlation Feature Selection
- PCA
- Attention Mechanism
- SHAP Explainability
